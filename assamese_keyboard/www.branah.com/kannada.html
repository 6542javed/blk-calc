This Kannada Keyboard enables you to easily type Kannada online without installing Kannada keyboard. You can use your computer keyboard or mouse to type Kannada letters with this online keyboard.

Pressing Esc on the Kannada keyboard layout will toggle the mouse input between virtual QWERTY keyboard and virtual Kannada keyboard. The key will also turn on/off your keyboard input conversion. Pressing Esc on your keyboard has the same function.

Press Shift or either Ctrl + Alt or AltGr for additional Kannada letters that are not visible on the keyboard.
