Cantonese to Jyutping converter

This Cantonese Keyboard enables you to easily type Cantonese online without installing Cantonese keyboard. You can use your computer keyboard or mouse to type Cantonese letters with this online keyboard.

As you type the phonetic reading of a letter, suggested letters appear on the number keys. Clicking on one of the suggested letters replaces the typed text with the letter chosen.

Use ◂ and ▸ keys to view the prev/next suggestions. The Space key can be used to select the first suggestion. Press Shift key for additional letters that are not visible on the keyboard.

Pressing Esc on the Cantonese keyboard layout will toggle the mouse input between virtual QWERTY keyboard and virtual Cantonese keyboard. The key will also turn on/off your keyboard input conversion. Pressing Esc on your keyboard has the same function.

For mobile phones and tablets, touch and hold inside the text area to copy the text. You can then paste the text in any app such as Facebook, Twitter, email, or search app.
